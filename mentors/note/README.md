# PHP & Laravel Backend Development Program

A comprehensive 12-week curriculum designed by GeekInk Innovation in collaboration with Aqtric Tech for aspiring backend developers.

## Program Overview
- **Duration**: 12 Weeks
- **Schedule**: 3 Days per Week
- **Class Length**: 2 Hours per Day
- **Teaching Days**: Day 1 & Day 2
- **Project Day**: Day 3 (Assignment & Project Presentation)

## Curriculum Structure

### [Week 1 – PHP Fundamentals I](week1/README.md)
- Day 1: PHP syntax, variables, data types, operators, and execution flow.
- Day 2: Conditional statements and loops with practical exercises.
- Day 3: Student Result Checker project presentation.

### [Week 2 – Functions, Arrays & Forms](week2/README.md)
- Day 1: Functions, parameters, return values, and reusability.
- Day 2: Arrays, PHP forms, and input sanitization.
- Day 3: Student Registration Form project presentation.

### [Week 3 – Sessions & Authentication](week3/README.md)
- Day 1: Sessions, cookies, and session security.
- Day 2: Password hashing, login and logout logic.
- Day 3: Pure PHP Authentication System presentation.

### [Week 4 – Database & PHP (PDO)](week4/README.md)
- Day 1: Database design concepts and MySQL fundamentals.
- Day 2: PDO connections, prepared statements, and CRUD operations.
- Day 3: Student Management System project presentation.

### [Week 5 – PHP Mini Project](week5/README.md)
- Day 1: Project architecture and code organization.
- Day 2: Error handling, pagination, and search.
- Day 3: Simple Blog System (Pure PHP) presentation.

### [Week 6 – Introduction to Laravel](week6/README.md)
- Day 1: Laravel overview, MVC architecture, and installation.
- Day 2: Folder structure, Artisan commands, and environment setup.
- Day 3: Laravel architecture and folders presentation.

### [Week 7 – Routing, Controllers & Blade](week7/README.md)
- Day 1: Routing, controllers, and request lifecycle.
- Day 2: Blade templating, layouts, and components.
- Day 3: Company Website (Laravel) presentation.

### [Week 8 – Database & Eloquent ORM](week8/README.md)
- Day 1: Migrations, models, and Eloquent ORM.
- Day 2: Relationships, seeders, and factories.
- Day 3: Blog Database with relationships presentation.

### [Week 9 – Authentication & Authorization](week9/README.md)
- Day 1: Laravel Breeze and authentication flow.
- Day 2: Middleware and role-based access control.
- Day 3: Multi-User Blog System presentation.

### [Week 10 – Validation, Files & Mail](week10/README.md)
- Day 1: Form validation and custom error handling.
- Day 2: File uploads and email notifications.
- Day 3: User Profile Management system presentation.

### [Week 11 – APIs & Security](week11/README.md)
- Day 1: REST API concepts and API routing.
- Day 2: Sanctum authentication and security best practices.
- Day 3: Laravel REST API tested with Postman presentation.

### [Week 12 – Capstone Project](week12/README.md)
- Day 1: Project planning, database design, and architecture review.
- Day 2: Implementation support and debugging.
- Day 3: Final project presentation and defense.

## Assessment & Evaluation
- **Weekly Projects and Assignments** – 60%
- **Final Capstone Project** – 30%
- **Attendance and Participation** – 10%

## Graduate Learning Outcomes
- Demonstrate strong understanding of PHP programming.
- Build secure and scalable Laravel applications.
- Design and manage relational databases.
- Develop RESTful APIs and implement security best practices.

## Getting Started
1. Clone this repository
2. Follow the weekly instructions in each week's directory
3. Complete all assignments and projects
4. Submit your work by the specified deadlines

## Prerequisites
- Basic knowledge of HTML, CSS, and JavaScript
- Familiarity with command line interface (CLI)
- PHP 8.0+ installed on your machine
- Composer for dependency management
- MySQL or MariaDB database server
- Code editor (VS Code, PHPStorm, etc.)
